# WMPlayer

## 这是什么
WMPlayer(Web Music Player)是一个基于HTML5的网页音乐播放器，其提供的诸多API和可定制的样式可以让你打造一个个性化的播放器。

## 特性
- 兼容IEX
- 提供丰富的API
- 可随意修改的HTML结构
- 丰富的主题
