# WMPlayer

这是V3版本开发的最新进展，距离开发完毕还剩3650天，请耐心等待
V2版本参见[Git@OSC](https://git.oschina.net/0936zz/WMPlayer)


如何开发
```bash
git clone https://github.com/0936zz/wmplayer.git
cd wmplayer
npm install -g gulp
npm install -g bower
npm install
bower install
npm run dev
```
如何构建源码
```bash
npm install -g gulp
npm install
npm run build
```

如何构建文档
```bash
npm install -g jsdoc
npm run doc
```
